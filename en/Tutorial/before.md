# Before You Begin

This is the quick start guide for Aiursoft Template. We all know that starting a new project can be a hassle, so we've prepared this tutorial to help you get up and running quickly.

This tutorial will take approximately 30 minutes. By the end, you'll have a fully functional project that you can use as a foundation for your future projects. For demonstration purposes, we'll build a simple note-taking app that converts Markdown to HTML, featuring multiple databases, RBAC, localization, OIDC integration, unit testing, and Docker deployment.

!!! info

    We strongly recommend that you do not skip any steps to ensure a complete experience.

Wishing you a smooth 30-minute adventure, and may you quickly become an Aiursoft Template master, fulfilling your web development dreams!
